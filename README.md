# üìö Intelligent Book Recommendation System using Text Mining and Machine Learning

## Overview

This project implements a Content-Based Book Recommendation System that recommends books based on textual similarity using Text Mining and Machine Learning techniques.

The recommendation engine analyzes book descriptions, genres, and author information to suggest similar books.

---

# Features

- Book Search
- Interactive Recommendation Interface
- Text Preprocessing
- TF-IDF Vectorization
- Bag of Words
- Cosine Similarity
- Content-Based Filtering
- Exploratory Data Analysis
- Interactive Jupyter Notebook UI using flask
